Technical Architecture

The Engine Room — Stability & Speed

  • Core Stack: Python 3.11 for ML, SQLAlchemy for data persistence, Streamlit for UI.
  • NLP Stack: FinBERT (Quantitative Sentiment) + Ollama/Mistral 7B (Narrative Reasoning).
  • ML Stack: XGBoost Ensemble models trained on 3 months of historical 15-min data.
  • Infrastructure: Containerized (Docker) for one-click deployment on any local machine.
